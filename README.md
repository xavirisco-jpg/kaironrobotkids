# Kairon Forest v1

A first real static prototype of the Kairon website: a calm, layered forest scene instead of a generic landing page.

## Upload to GitHub Pages
Upload the **contents** of this folder to the root of your GitHub repository:

- `index.html`
- `style.css`
- `script.js`
- `assets/`

Do not upload the ZIP itself and do not place the contents inside an extra nested folder.

## One thing to edit before public launch
Open `index.html`, search for `YOUR_YOUTUBE_CHANNEL_URL`, and replace it with the exact Kairon YouTube channel URL.

## Current features
- Original local SVG Kairon character and layered forest assets
- Responsive mobile and desktop layout
- Gentle pointer parallax, clouds, fireflies and Kairon movement
- Reduced-motion support
- GitHub Pages friendly — no framework or build step
- A visual message form (not connected to email yet)
