# Kairon Robot Kids — website v2

This is a static GitHub Pages website: no build tools, no installs, no nonsense.

## Put it live on GitHub Pages

1. Download and unzip this project.
2. In the GitHub repository, delete the old web files (not the `.git` folder if you are using Git locally).
3. Upload the contents of this folder: `index.html`, `privacy.html`, `terms.html`, and the whole `assets` folder.
4. Commit the changes to the `main` branch.
5. Wait 1–3 minutes and refresh your custom domain.

## Two small things to personalize

- **Contact email:** edit `assets/script.js`, find `const INBOX_ADDRESS`, and replace `hello@kaironrobotkids.com` with the real mailbox once it exists.
- **Social links:** edit the three links near the bottom of `index.html` (YouTube, Instagram, TikTok). The YouTube URL is already set to `https://www.youtube.com/@KaironRobotKids`.

## What it includes

- Responsive desktop/mobile landing page
- English / Spanish language toggle
- YouTube call-to-action
- Functional contact flow that opens the visitor's email app (works with static GitHub Pages hosting)
- Privacy Policy and Terms pages

## Important legal note

The privacy and terms pages are a practical starter draft for this exact static site. Before enabling analytics, ad pixels, embedded forms, newsletters, or collecting data directly, update the policy and get legal review for your final setup.
